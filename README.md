Password Checker 🔑
This is a simple Python project that helps users create a strong password.
What does it do?
It checks if your password has:
• 8 or more characters.
• Uppercase & Lowercase letters.
• Numbers.
•Special symbols (like @, #, $).
How to use it:
1.	Run the code.
2.	Enter a password.
3.	If it’s weak, the program will tell you what’s missing.
4.	You have 3 tries to get it right!
